# Day 2: Interfaces and Cables 🔌

## Objectives
* Identify different types of network cables and connectors (RJ-45, Fiber).
* Understand Ethernet standards (IEEE 802.3).
* Configure connections based on device types (Straight-through vs Crossover).

## Key Takeaways
* **Copper Cabling:** UTP (Unshielded Twisted Pair) uses RJ-45. 10BASE-T/100BASE-T use 2 pairs, while 1000BASE-T uses all 4 pairs.
* **Fiber Optic:** * *Single-mode:* Long distance, laser-based, narrow core.
    * *Multi-mode:* Shorter distance, LED-based, wider core.
* **Auto MDI-X:** A feature that automatically detects the required cable type, though knowing the manual rules is essential for CCNA.

## Lab: Connecting Devices
Successfully built a topology requiring specific cable choices:
* Fiber for long-distance router links (3km).
* Crossover cables for similar devices (Switch-to-Switch, Router-to-Router).
* Straight-through for dissimilar devices (PC-to-Switch).
